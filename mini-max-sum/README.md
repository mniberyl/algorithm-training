# 배열 요소의 최소, 최대 합계 구하기

배열 안에 n개의 양의 정수가 주어지면 그 중 n-1개의 정수를 더해 최소값과 최대값을 구합니다. 그리고 각각의 최소값과 최대값을 공백으로 구분하고, long 타입의 정수로 출력합니다. 예를 들어 arr = [1, 3, 5, 7, 9] 일 때,

+ 최소값은 1 + 3 + 5 + 7 = 16,<br>
+ 최대값은 3 + 5 + 7 + 9 = 24 입니다.
<br>

따라서, 최소값 16과 최대값 24가 반환됩니다.
<br>
<br>

input
```
1 2 3 4 5
```


output
```
10 14
```
