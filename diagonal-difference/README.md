# 대각선 합의 차이 구하기

정사각형 행렬의 대각선 합계 간의 차이를 구합니다. 예를 들어 정사각형 행렬 arr이 아래와 같을 때,<br>
1  2  3<br>
4  5  6<br>
9  8  9<br>
왼쪽에서 오른쪽 대각선의 합은 1 + 5 + 9 = 15. 오른쪽에서 왼쪽 대각선의 합은 3 + 5 + 9 = 17입니다. 이 합계 간의 차이는 절대값으로 구합니다. 따라서, |15 - 17| = 2 이므로 정수 2를 반환합니다.

<br>

input
```
11 2 4
4 5 6
10 8 -12

```


output
```
15
```
