# 배열 요소끼리의 비교

길이가 같은 배열의 요소를 비교해서 점수를 부여하는 과제가 있습니다. Alice와 Bob은 이 과제에 참여했습니다. Alice의 배열 a는 [1, 2, 3]이고 Bob의 배열 b는 [3, 2, 1]이라고 할 때, 
+ a[1] < b[1], Bob이 1점 얻습니다.
+ a[2] = b[2], 두 사람 모두 점수가 없습니다.
+ a[3] > b[3], Alice가 1점 얻습니다.

따라서, [1, 1]의 배열을 반환합니다.

<br>

input 1
```
5 6 7 
3 6 10
```


output 1
```
1 1
```

<br>


input 2
```
17 28 30 
99 16 8
```

output 2
```
2 1
```
